主要風險與對策
!
Supabase 閒置自動暫停
Free tier 7天無活動自動暫停，家長打開 App 看到錯誤，體驗極差。對策：用 cron-job.org（免費）每3天 ping 一次 /api/health endpoint，Claude Code 可在5分鐘內幫你加好。
~
Storage 超出 1GB 上限
約 80 位家長時觸頂。對策：老師審核批准後自動刪除原圖，只保留 JSON，可延長至約 300 位家長。到時 Supabase Pro $25/月 = HK$195，遠低於收入。
~
Claude Code 是持續性成本
$100 USD/月很重，佔初期成本 83%。建議：App 穩定後降至 Free tier（每月 2000 messages），只在開發新功能時才升回 Pro。
i
AI API 成本難以預測
每位家長估計 HK$2/月，但大量 OCR 或 Variation 生成可能更高。對策：在 App 設每位家長每月最多上載 4 份 Past Paper，限制 Variation 生成次數，即可有效控制。
